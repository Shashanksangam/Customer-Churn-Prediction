# Customer Churn Prediction System

A Machine Learning web application built using Streamlit to predict whether a customer is likely to leave a telecom service.

## Features

- Logistic Regression
- Decision Tree
- Random Forest
- Customer probability prediction
- Risk classification
- Prediction history
- Analytics dashboard
- Feature importance visualization
- Download prediction reports
- Multiple model selection

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib
- Joblib

## Dataset

Telco Customer Churn Dataset
